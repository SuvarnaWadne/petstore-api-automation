# PetStore API Automation Framework

This is a Java + REST Assured + TestNG automation framework created for testing the Swagger PetStore APIs.

## Tech Stack
- Java 8+
- Maven
- REST Assured
- TestNG
- Hamcrest Matchers
- Jenkins (CI/CD)

---

## How to Run Tests
### Option 1 — Eclipse/IntelliJ
Right-click `testng.xml` → **Run as TestNG Suite**

### Option 2 — Maven CLI
```bash
mvn clean test
